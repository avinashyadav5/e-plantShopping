# Paradise Nursery Shopping Application

This is the final project for the "Paradise Nursery" shopping application. It features a complete shopping cart workflow built using React and Redux Toolkit.

## Features

*   **Landing Page**: A welcoming landing page featuring a beautifully styled background, an "About Us" section detailing our mission, and a "Get Started" button linking to the product catalog.
*   **Product Listing Page**: Displays a selection of lush houseplants categorized into Aromatic, Medicinal, and Decorative plants. Each plant can be added directly to your cart.
*   **Shopping Cart Page**: A dynamic shopping cart that tracks selected plants. It displays the unit cost, total item cost, and grand total. Users can increment, decrement, or completely delete items from the cart.
*   **Dynamic Navigation**: A top navigation bar (Header) that shows the total number of items in your cart and provides easy navigation between the "Plants" catalog, the "Cart", and the Home page.
*   **State Management**: Uses Redux Toolkit to manage global state securely and efficiently across the application.

## Technologies Used

*   **React**: For building the user interface.
*   **Redux Toolkit & React-Redux**: For managing the shopping cart state.
*   **Vite**: Fast frontend build tool.
*   **Vanilla CSS**: Used for styling with modern features like CSS variables and flex/grid layouts.

## Running Locally

To run the application locally on your machine, follow these steps:

1.  Clone this repository to your local machine.
2.  Navigate to the project directory: `cd e-plantShopping`
3.  Install dependencies: `npm install`
4.  Start the development server: `npm run dev`
5.  Open your browser to the local URL provided in the terminal.
